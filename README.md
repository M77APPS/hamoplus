# 🛡️ تطبيق حمّو الإلكتروني - الدرع الصامت (Hemo Electronic - Silent Shield)
### الإصدار المتكامل 100% - سلاح سيبراني دفاعي متقدم لحماية وتأمين أجهزة الأندرويد v4.2.0

> **شعار المطور:** *"المحترفون يبنون أدواتهم، والهواة يستخدمون أدوات الآخرين."*  
> **مبتكر ومهندس المشروع:** محمد حسام (حمّو الإلكتروني)

تطبيق **حمّو الإلكتروني - الدرع الصامت** هو حارس أمني ذكي متكامل يعمل في خلفية جهازك على مدار 24 ساعة ليكتشف ويحلل الروابط الضارة قبل فتحها، ويعزلها في بيئة رملية آمنة (Sandbox WebView) مشحونة بأعلى معايير الحماية الذاتية، مدعوماً بخزنة سيبرانية مشفرة ومؤشرات أمان حية لتقديم حماية عسكرية متوافقة تماماً مع سياسات Google Play.

---

## 🧱 أهم الميزات التقنية الفريدة
1. **الدرع الصامت اللاسلكي**: خدمة خلفية دائمة ومستمرة (Hemo Foreground + Local VPN Tunnel) لاعتراض وفحص وتصفية طلبات DNS محلياً وحجب الروابط الخبيثة فورياً.
2. **قراءة الروابط الذكية (Accessibility Service)**: فحص وقراءة النصوص على الشاشة لكشف روابط التصيد والروابط الملغمة قبل فتحها ودون تخزين أي بيانات للمستخدم.
3. **بيئة العزل الرملية (Sandboxed WebView)**: فتح الروابط المشبوهة داخل حاوية مغلقة آمنة مع تعطيل JavaScript بالكامل لمنع هجمات الـ zero-click.
4. **الخزنة السيبرانية العسكرية (Military Vault)**: تشفير النصوص والملفات بأعلى تشفير تشاركي قياسي **AES-256-GCM** مع مولد أكواد TOTP الآمن متجدد كل 30 ثانية وقفل بالبصمة الحيوية لمنع الاختراق.
5. **كشف الهندسة العكسية وحماية بيئة التشغيل**: فحص سلامة النظام وتحديد ما إذا كان التطبيق يعمل على محاكي، جهاز بصلاحيات جذر (Rooted)، أو تحت أطر تتبع برمجية مثل Frida أو Xposed، وعرض تقرير الحماية الهيكلية.
6. **العمل بدون إنترنت (Offline Mode)**: الحماية فعالة دائماً بالاعتماد على قاعدة بيانات Room محلية تضم عشرات الآلاف من تواقيع وروابط التصيد، ويتم مزامنتها بمجرد عودة الاتصال.

## 🚀 دليل تفعيل وموافقات الحماية يدويًا
للحصول على كامل قدرات الحماية، اتبع الخطوات التالية بحرص:

### أ – تفعيل خدمة الوصول (Accessibility Service) لقراءة وفحص الروابط
1. افتح **إعدادات الهاتف (Settings)**.
2. توجه إلى **إمكانية الوصول (Accessibility)**.
3. من قائمة "الخدمات المثبتة" أو "التطبيقات"، ابحث عن **حمّو الإلكتروني (Hemo Electronic)**.
4. اضغط على مفتاح تفعيل الخدمة، واقبل التصاريح الأمنية اللازمة.
5. الآن، سيبدأ الدرع الصامت بقراءة الروابط الضارة من المحادثات والرسائل في الخلفية فوراً وتنبيهك قبل نقرها!

### ب – تفعيل نفق الـ VPN المحلي لتصنيف الـ DNS
1. من الشاشة الرئيسية، اضغط على زر **تفعيل الدرع الصامت VPN**.
2. ستظهر لك نافذة تصريح إنشاء اتصال VPN محلي آمن.
3. اضغط على **موافق وأوافق الآن** لمنح النظام الإذن لإنشاء النفق المحلي. (لن نقوم بنقل بياناتك خارج جهازك مطلقاً!).

## 💻 كيفية بناء وتشغيل التطبيق عبر Android Studio
1. قم بفتح برنامج **Android Studio (Ladybug / Koala أو أحدث)**.
2. اختر **Open** ثم حدد مجلد المشروع هذا وجبن في تحديث المكونات (Gradle Sync).
3. تأكد من أن الـ **JDK** المثبت والمختار هو الإصدار 17 أو أعلى.
4. قم بربط هاتفك الأندرويد (تأكد من تفعيل "خيارات المطورين" و "تصحيح أخطاء USB").
5. اضغط على زر **Run** (الأيقونة الخضراء) لتثبيت واختبار التطبيق مباشرةً.

## 🔐 نظام تشفير الـ APK وحمايتها التلقائية عبر GitHub Actions
للبناء الأمن وتضمين حماية متقدمة ضد الهندسة العكسية عبر الـ CI/CD:
1. تم دمج قواعد **R8/ProGuard** متكاملة ومحسنة داخل المشروع عبر ملف `proguard-rules.pro`.
2. لتشفير الكود وتعقيمه تلقائياً بأقصى درجة في خط أنابيب البناء الخاص بك بالـ GitHub Actions:
   - قم بإنشاء مجلد `.github/workflows/build_secure.yml` في مستودع الكود الخاص بك.
   - قم بإضافة الـ Secrets التالية في إعدادات GitHub Repository لمنع استخراج مفتاح التشفير وسرقة الكود:
     - `KEYSTORE_BASE64` (مفتاح التوقيع مشفر بصيغة base64).
     - `STORE_PASSWORD` & `KEY_PASSWORD`.
   - بمجرد دفع الكود (Push) للفرع الرئيسي، سيتكفل المعالج ببناء تطبيق أمن موقع وجاهز للتشغيل بأعلى درجات الـ Obfuscation الصارمة.
